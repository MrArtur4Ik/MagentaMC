# MagentaMC
Minecraft Classic 0.30 server written on Python.

First type ```python3 -m venv .venv``` to create virtual environment.

Second type ```source .venv/bin/activate``` to activate environment (for Linux)
For Linux in fish terminal: ```source .venv/bin/activate.fish```
For PowerShell in Windows: ```.\.venv\Scripts\Activate.ps1```

Then type ```pip3 install -r requirements.txt``` to install requirements.

And start server: ```python3 start.py```
